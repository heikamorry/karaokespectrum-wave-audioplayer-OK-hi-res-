# karaokespectrum-wave-audioplayer-OK-hi-res-
A Qt 6/C++ desktop music player with lyric sync, karaoke highlighting, and real-time audio spectrum visualization/基于 Qt 6/C++ 的桌面音乐播放器，支持歌词同步、KTV 高亮和实时音频频谱可视化
这是一个基于 Qt 6 / C++ 开发的桌面音乐播放器，集成了 QMediaPlayer 音频播放、播放列表管理、专辑封面显示、进度与音量控制，并支持 LRC 歌词同步显示、KTV 卡拉 OK 高亮效果 以及基于 QAudioBufferOutput + kissfft 的实时音频波形与频谱可视化。
This is a desktop music player built with Qt 6 / C++, featuring QMediaPlayer-based audio playback, playlist management, album cover display, progress and volume control, as well as LRC lyric synchronization, karaoke-style highlighting effects, and real-time audio waveform and spectrum visualization powered by QAudioBufferOutput + kissfft.
